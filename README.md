Smart Surveillance & Attendance System — Final Year Project: Full-Stack Agentic AI System
Built a complete AI-powered security and attendance platform from the ground up. The core is a face recognition engine using ArcFace embeddings and cosine similarity — hitting 95% accuracy on live feeds without any offline retraining. New users register directly through the running app via webcam or photo upload, with augmentation and duplicate checks handled automatically. Intelligence layer runs two Groq-hosted agents: one watches the feed continuously and fires classified threat alerts the moment an unknown face appears, the other lets administrators query attendance records in plain English and get instant answers. Everything surfaces through a Streamlit dashboard built to be usable by non-technical staff.
